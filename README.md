# students

Здесь лежат учебные проекты. Каждый — отдельная папка со своими зависимостями и инструкциями:

| Папка          | Что это |
| -------------- | ------- |
| [`clientserver`](clientserver/) | Клиент (React + Vite) и сервер (Express + TypeScript), npm workspaces. Подробности в [`clientserver/README.md`](clientserver/README.md). |
| [`rpgsaga`](rpgsaga/) | Проект на TypeScript. Подробности в [`rpgsaga/README.md`](rpgsaga/README.md). |

Откройте нужную папку, затем выполните `npm install` и команды из её README.
